# تشغيل الخادم

## الحل النهائي لمشكلة vite

اختر إحدى الطرق التالية لتشغيل الخادم:

### الطريقة الأولى (الأفضل):
```bash
node npm-run-dev.js
```

### الطريقة الثانية:
```bash
node vite-fix.js
```

### الطريقة الثالثة:
```bash
npx vite --host 0.0.0.0 --port 8080
```
